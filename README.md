# Portfolio V1 — Faizan Khan

A highly interactive and visually engaging personal portfolio website. Built with an emphasis on a cinematic user experience, utilizing modern web technologies, 3D elements, and advanced animations.

## 🔗 Live Site
[faizan-khan.vercel.app](https://faizan-khan.vercel.app)

## 🛠️ Built With

### Core
- React 18 + TypeScript
- Vite
- Tailwind CSS

### 3D & Animations
- **Three.js** / **React Three Fiber** / **Drei** (For 3D elements and interactive models like the MacBook showcase)
- **GSAP** (GreenSock Animation Platform for robust timeline animations)
- **Framer Motion** (For fluid layout transitions and micro-interactions)
- **Matter.js** (For 2D physics simulations)

### UI & Icons
- Lucide React

## 📂 Sections
- **Hero** — Splash landing with animated signature and GSAP intro
- **About / Skills** — Background information and tech stack overview
- **Projects** — Featured work with interactive 3D and glassmorphism cards
- **Research** — Academic work displayed with premium aesthetics 
- **Work Experience** — Professional roles and timeline
- **Gallery** — Masonry grid showcasing events, hackathons, and milestones
- **Contact** — Connectivity links and social presence

## 🚀 Run Locally
```bash
# Clone the repository
git clone https://github.com/faizankhan1309/Portfolio_V1.git

# Navigate into the directory
cd Portfolio_V1

# Install dependencies
npm install

# Start the development server
npm run dev
```
